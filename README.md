# HackMatch AI

AI-based Hackathon Teammate Finder.

## Features
- Skill-based semantic matching using Sentence Transformers
- Cosine similarity scoring
- Automatic team formation using KMeans clustering
- FastAPI backend
- Simple frontend interface

## Tech Stack
- Python
- FastAPI
- sentence-transformers
- scikit-learn
- HTML, CSS, JavaScript

## Run Backend
cd backend
uvicorn app.main:app --reload

## Open Frontend
Open frontend/index.html in browser
